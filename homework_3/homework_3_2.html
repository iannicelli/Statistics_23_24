A probability space is a mathematical model used to describe a random experiment. It is defined by a triple (Ω,F,P)(\Omega, \mathcal{F}, P), where:
  1. Ω\Omega is the sample space, a set of all possible outcomes of the experiment.
  2. F\mathcal{F} is the event space or σ\sigma-algebra, a set of events where each event is a set of outcomes from Ω\Omega. It must satisfy three conditions:
    • It contains the empty set.
    • If an event E is in F\mathcal{F}, then its complement is also in F\mathcal{F}.
    • If events E1,E2,E3,...E_1, E_2, E_3, ... are in F\mathcal{F}, then their union is also in F\mathcal{F}.
  3. PP is the probability measure, a function that assigns to each event in F\mathcal{F} a probability, which is a number between 0 and 1. It must satisfy three conditions:
    • P(∅)=0P(\emptyset) = 0 and P(Ω)=1P(\Omega) = 1.
    • P(E)≥0P(E) \geq 0 for every event E in F\mathcal{F}.
    • If E1,E2,E3,...E_1, E_2, E_3, ... are disjoint events in F\mathcal{F} (i.e., no two of them have common outcomes), then P(⋃i=1∞Ei)=∑i=1∞P(Ei)P(\bigcup_{i=1}^{\infty} E_i) = \sum_{i=1}^{\infty} P(E_i).
For example, consider a simple experiment of flipping a fair coin once. The probability space for this experiment would be:
  • Ω={"Heads","Tails"}\Omega = \{\text{"Heads"}, \text{"Tails"}\}: The sample space consists of two outcomes: "Heads" and "Tails".
  • F={∅,{"Heads"},{"Tails"},{"Heads","Tails"}}\mathcal{F} = \{\emptyset, \{\text{"Heads"}\}, \{\text{"Tails"}\}, \{\text{"Heads"}, \text{"Tails"}\}\}: The event space consists of four events: no outcome (empty set), "Heads", "Tails", and both "Heads" and "Tails".
  • PP: The probability measure assigns a probability of 0.5 to both "Heads" and "Tails", 0 to the empty set, and 1 to the set containing both "Heads" and "Tails".
For Exercise 1, we can define the probability space as follows:
  1. Sample Space (Ω)(\Omega): This is the set of all possible outcomes of an attack on a system. In this case, it would be Ω={−1,1}\Omega = \{-1, 1\}, where -1 represents a successful penetration (attack) and 1 represents a successful defense (shield).
  2. Event Space (F)(\mathcal{F}): This is the set of all possible combinations of outcomes for the N attacks on the M systems. It's a bit complex to list out all possibilities, but it would include sequences like (-1, -1, 1, ..., -1) representing the outcomes of N attacks on a system.
  3. Probability Measure (P)(P): This is a function that assigns a probability to each event in F\mathcal{F}. Given the constant penetration probability p for each attack, we can define P as follows: P(-1) = p (probability of penetration) and P(1) = 1 - p (probability of successful defense).
